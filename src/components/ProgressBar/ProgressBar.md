Example of a default progress bar
```vue
<template>
	<progress-bar class="small" :value="50">
		Text
	</progress-bar>
</template>
```


Example of a progress bar with its value hidden
```vue
<template>
	<progress-bar class="small" :hide-value="true" :value="50">
		Text
	</progress-bar>
</template>
```
